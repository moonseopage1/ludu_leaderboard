# Ludu Leaderboard

Simple Ludu leaderboard app using:

- HTML
- Tailwind CSS CDN
- JavaScript
- Node.js Express
- TXT file storage

## Install

```bash
npm install
```

## Run

```bash
npm start
```

Open:

```txt
http://localhost:4000
```

## TXT Database

All data is saved in:

```txt
ludu-data.txt
```

## Scoring

```txt
1st = 4 points
2nd = 3 points
3rd = 2 points
4th = 1 point
```
