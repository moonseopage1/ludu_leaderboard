# Ludu Leaderboard

Simple Ludu leaderboard app using:

- HTML
- Tailwind CSS CDN
- JavaScript
- Node.js Express
- JSON file storage locally
- JSON file storage on Vercel

## Install

```bash
npm install
```

## Run

```bash
npm start
```

Open:

```txt
http://localhost:4000
```

## Data Storage

Local development saves data in:

```txt
ludu-data.json
```

Local and Vercel API routes save data in `ludu-data.json`. See `VERCEL_DEPLOYMENT.md`.

## Scoring

```txt
1st = 4 points
2nd = 3 points
3rd = 2 points
4th = 1 point
```
